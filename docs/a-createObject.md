# Use Case A. Creating an object
## Case
1. 使用者於任一模式下在 Rect 或 Oval 按鈕範圍內按下 mouse 的左鍵，但是不放開 (mouse pressed)。
2. 原模式按鈕的顏色復原，目標 Rect 或 Oval 按鈕的顏色變黑，以告知使用者目前欲建立物件的 mode。
3. 使用者不放開左鍵，進行拖曳(drag)的動作。
4. 使用者拖曳至編輯區域的任一範圍內，放開左鍵 (mouse released) 。
5. 在編輯區內，繪製一個空白的 Rect 或 Oval 物件。
6. 目標 Rect 或 Oval 按鈕的顏色復原，原模式按鈕的顏色變黑（選取的模式回到本Case 以前選取中的模式）。
