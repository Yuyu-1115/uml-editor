# Use Case D. Group / Ungroup objects
## Precondition
適用於按鈕 select 被按下的情況。
## Case 1:
1. 當有大於 2 (含)個物件處於被 select 的狀態時。
2. 使用者到 Edit Menu 點選 Group 選項。
3. 處於被選取狀態的基本物件將合併成一個 composite 物件。
## Case 2
1. 當唯一 1 個 composite 物件處於被 select 的狀態時。
1. 使用者到 Edit Menu 點選 Ungroup 選項。
3. Composite 物件解構最外一層。
### Alternatives D.1
1. 當只有 1 個物件被選取時，點擊 Group 選項不會有任何動作。
### Alternatives D.2
1. 當大於 2 個物件被選取時，點擊 Ungroup 選項不會有任何動作。
