# Use Case G. Customize Label Style
## Precondition
當某一個基本物件處於被 select 的狀態時。
## Case
1. 使用者到 Edit Menu 點選 Label 項目。
2. 彈跳視窗 Customize Label Style，提供以下選項，並預設顯示標籤原設定：
a. 標籤名稱 (Label Name)：輸入新名稱，變更顯示標籤內容。
b. 標籤顏色 (Label Color)：變更標籤背景色，更變方式不限。
3. 使用者按下 OK，Object 上的 label 即時更新。
