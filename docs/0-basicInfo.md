# Basic Info
本 Workflow Design 編輯器總共分成 3 個部分，第一個部分是編輯的區域稱之為 Canvas
area 也就是所謂的畫布。編輯區域左邊有一排按鈕。你要完成的按鈕最少如下 6 個按鈕。
從上到下分別是：
- Select（預設選取）
- Association
- Generalization
- Composition
- Rect
- Oval

## Scnearios
本需求文件採用 Object & Link 來描述系統需求與規格。我們將使用此 Workflow Design
editor 的 scenarios 先分類成幾大類：
A. 建立物件
B. 連結物件
C. 選取物件
D. 群組物件
E. 移動物件
F. 調整物件大小
G. 自定義物件
